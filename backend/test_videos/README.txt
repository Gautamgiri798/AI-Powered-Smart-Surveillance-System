# 🧪 SentinelVision — Video Testing Mode

This folder is used to simulate surveillance scenarios using pre-recorded video files.

### 🚀 How to use Testing Mode:
1.  Place any `.mp4` or `.avi` surveillance clip in this folder.
2.  Rename your file to **`sample.mp4`** (or change the path in the dashboard).
3.  On the **SentinelVision Dashboard**, select the **"Testing Mode"** camera.
4.  Click **"Start Engine"**.

### ✨ Highlights:
- **Infinite Looping:** The AI will automatically reset to the first frame when the video reaches the end.
- **HUD Mode Indicator:** The live feed will display **"MODE: TESTING"** to distinguish it from live hardware.
- **Accurate Benchmarking:** Use this to verify how the AI identifies weapons or behaves with specific loitering scenarios!
